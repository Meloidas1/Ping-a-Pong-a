🏓 Pong Game (Python + Pygame)

This is a simple 2D Pong game built using Pygame and Python.
The game is a classic arcade-style competition between two players where each controls a paddle to bounce a ball and score points.

🎮 Gameplay

This project recreates the classic Pong experience:

Two players control paddles on opposite sides
The goal is to prevent the ball from passing your paddle
First player to reach 10 points wins
🧠 Features
2-player local gameplay
Smooth paddle movement
Collision detection (ball & paddles/walls)
Score system
Win screen
Restart system (press R)
Simple game loop using FPS control
🎯 Controls
Player 1 (Left Paddle)
W → Move Up
S → Move Down
Player 2 (Right Paddle)
↑ Arrow → Move Up
↓ Arrow → Move Down
Game Controls
R → Restart game
Close window → Quit game
